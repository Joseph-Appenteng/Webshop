# Webshop

Zoals de naam al een klein beetje verklapt gaan we dit keer de front-end van een webshop maken. Dit doen wij op basis van een zelfgekozen onderwerp (games, sierraden, parfum, elektrische grasmaaiers, noem het maar op!) en een toegewezen stijl. Ik ga voor een Game webshop maken met Mediamarkt als een togewezen stijl. We werken dit project op basis van een design en een product backlog.
